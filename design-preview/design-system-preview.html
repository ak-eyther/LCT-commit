Design System Summary

Key decisions and rationale

✅ No Purple OR Blue: Replaced #9333ea and minimized blue - using slate/gray tones instead.

✅ Compact & Efficient: Smaller buttons (6px 12px), compact UI inspired by X.com and Grok.

✅ Consistent Spacing: 8px base scale (8px, 16px, 24px, 32px).

✅ Accessible: WCAG 2.2 compliant color contrasts.

✅ shadcn/ui Principles: CSS variables, modern approach, easy theming.
